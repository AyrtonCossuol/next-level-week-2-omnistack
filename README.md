# Tecnologias
Esse projeto foi feito utilizando as seguintes tecnologias:

* [Typescript](https://www.typescriptlang.org/)      
* [Node.js](https://nodejs.org/en/)      
* [React](https://reactjs.org/)      
* [Expo](https://expo.io/)       
* [Express](https://expressjs.com/) 

# Funcionalidades

* Site para cadastro de estudante e professores.
* App para conectar estudantes e professores.

# Como rodar
```bash
# Clone o Repositoria
$ git clone https://github.com/AyrtonCossuol/next-level-week-2-omnistack.git
```
### Rode a API

```bash
# Vá para a pasta do servidor
$ cd next-level-week-2-omnistack/server

# Instale as depedencias
$ yarn install

# Rode a aplicação
$ yarn start
```

### Rode o Project Web

```bash
# Vá para a pasta web
$ cd next-level-week-2-omnistack/web

# Instale as depedencias
$ yarn install

# Rode a aplicação
$ yarn start
```

### Run Mobile Project
Para rodar o projeto mobile você precisa de um celular com o [expo](https://play.google.com/store/apps/details?id=host.exp.exponent) instlado ou um emulador android/ios.

```bash
# Vá para a pasta mobile
$ cd next-level-week-2-omnistack/mobile

# Instale as depedencias
$ yarn install

# Rode a aplicação
$ yarn start
```
Depois leia o QRCode com o app do [expo](https://play.google.com/store/apps/details?id=host.exp.exponent) ou rode em um emulador.
